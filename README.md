# Зал: A/B/C

Мобильная веб-версия дневника тренировок под текущую программу A/B/C.

## Запуск

```powershell
cd "C:\Users\maksd\OneDrive\Документы\New project\gym-training-app"
npm start
```

После запуска открой:

```text
http://localhost:4173
```

## Что уже есть

- расписание A/B/C по понедельникам, средам и пятницам;
- ввод веса, подходов и повторов с телефона;
- локальное хранение истории в IndexedDB браузера;
- история тренировок;
- простой график прогресса по каждому упражнению;
- осторожная автопрогрессия без автоматического наказания за неудачу;
- вопрос о причине, если тренировка не удалась;
- экспорт истории в CSV.

## Логика безопасности

Приложение адаптировано под осторожный режим тренировок: нейтральная шея, спокойное дыхание, отсутствие рывков и отказа. Вес повышается только после легкой тренировки без боли, головокружения, онемения и прострела. При плохом самочувствии, температуре, одышке, боли в груди или выраженной слабости тренировку лучше пропустить и обсудить нагрузку с врачом.
